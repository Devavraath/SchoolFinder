Description:

The School Finder Web App is a powerful tool designed for Government school teachers during their Transfer application process. It offers a comprehensive set of features to assist teachers in the following ways

    1. Easily find school locations via Google Maps
    2. Measure distance from their Home Location
    3. Sort multiple school locations by ascending distance
    4. View the Location directly in Google Maps
    5. Save the selected list of items in an Excel File
Traditionally, this task has been a time-consuming manual process involving:
Manually searching and locating school addresses on Google Maps - Inputting data into spreadsheets - Capturing route maps as screenshots and embedding them in PDFs - Compiling all the information into a readable format.
Last year, to validate 100+ school locations for my Mother, it took 3 people & 5 hours to do all these.
So, 12 hours, an all-nighter and lots & lots of prompt engineering later....
I present to you "The School Finder Web App."
